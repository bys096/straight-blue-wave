## 설치 및 실행방법
1. git clone
2. 각 서버 npm i (node 서버 동작 오류 시 babel 추가
```
npm i @babel/node -g
```
3. 노드 서버 기동
```
 npm run dev
```
4. localtunnel 기동(node port번호 4000을 따라감)
```
npx localtunnel --port 4000
```
5. 발급 받은 URL로 build 폴더의 socket.io에 관한 script 태그 수정
6. URL로 접속
